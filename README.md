# Pomo Focus

A simple Pomodoro timer web application built to help improve focus and productivity.

## Live Demo
https://pomo-focus-seven.vercel.app

## Features
- Pomodoro timer (work and break sessions)
- Clean and minimal UI
- Session tracking
- Basic productivity dashboard

## Tech Stack
- React
- Vite
- Tailwind CSS
- Chart.js
- Lucide React

## Getting Started

### Clone the repository
git clone https://github.com/your-username/pomo-focus.git

### Install dependencies
npm install

### Run Locally
npm run dev

### Build for production
npm run build

## Purpose
Built as a practice project to improve frontend development skills, especially working with timers, state management, and UI design.
